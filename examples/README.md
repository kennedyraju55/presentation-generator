# Examples for Presentation Generator

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`load_config()`** — Call load_config
- **`get_formats()`** — Call get_formats
- **`get_slide_templates()`** — Call get_slide_templates
- **`get_visual_suggestions()`** — Call get_visual_suggestions
- **`estimate_timing()`** — Estimate presentation timing.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
